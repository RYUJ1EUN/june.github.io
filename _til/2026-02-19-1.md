#### DevOps: Development + Operations

개발-운영 간 격차 감소 $\to$ 서비스의 안정적 제공

##### DevOps 무한 루프

$\hookrightarrow \text{Plan}\ \underset{\text{DEV} }{\underbrace{\to \text{Code} \to \text{Build} \to \text{Test} \to}}\ \text{Release} \ \underset{\text{OPS} }{\underbrace{\to \text{Deploy} \to \text{Operate} \to \text{Monitor} \to }}$

##### 핵심 기술: 자동화 & 피드백

CI/CD, 구성 관리, 컨테이너 기술, 인프라 자동화(IaC), 모니터링, 로깅 시스템 등

#### DevSecOps: DevOps + Security

취약성 분석과 보안 교육 추가 수행

- 정적 애플리케이션 보안 테스트(SAST) 도구: 개발 단계에서 소스 코드를 자동 분석하여 보안 취약점 가능성 조기에 탐지
- 소프트웨어 구성 분석(SCA) 도구: 빌드 과정에서 사용 중인 오픈 소스 및 외부 라이브러리의 취약점과 라이선스 준수 여부를 자동 점검
- 대화형 애플리케이션 보안 테스트(IAST) 도구: 애플리케이션 실행 중 내부 동작 모니터링, 런타임 환경에서 발생하는 보안 문제 탐지
- 동적 애플리케이션 보안 테스트(DAST) 도구: 실행 중인 애플리케이션을 외부에서 테스트, 실제 공격자 관점에서 보안 취약점 검증
